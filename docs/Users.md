# Users

## Create user
To add a user, open Admin → Users → Invite user and enter their email.
The invited user receives an email and sets their own password on first login.

## Delete user
To delete a user, open Admin → Users, select the user and click Delete.
Deletion is permanent; the user's personal data is fully removed after 30 days.

## Roles and permissions
Each user has a role: Owner, Admin, Member or Viewer.
Only Owners and Admins can invite or delete other users.
